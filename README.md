# InfoProgramming

Selamat datang di InfoProgramming, repositori dokumentasi komprehensif yang membahas berbagai topik
pemrograman dan administrasi sistem. Sumber daya ini menyediakan panduan praktis dan lembar
contekan untuk pengembang dan administrator sistem.

## Tentang Proyek Ini

Situs dokumentasi ini berisi kumpulan tutorial, panduan, dan materi referensi yang berfokus pada:

- Administrasi sistem Linux
- Pengaturan dan konfigurasi pemrograman
- Alat-alat pengembangan dan utilitas

## Daftar Isi

Dokumentasi ini diorganisir menjadi beberapa bagian:

### Bagian 1

- [Upgrade LibreOffice](Bagian/-UpgradeLibreOffice.md) - Panduan untuk meng-upgrade LibreOffice
  pada sistem Linux
- [Ghostscript Cheat Sheet](Bagian/-GhostscriptCheatSheet.md) - Referensi cepat untuk
  perintah-perintah Ghostscript
- [EditorConfig Guide](Bagian/-EditorConfigGuide.md) - Panduan konfigurasi untuk EditorConfig
- [Naming Files and Directories](Bagian/-NamingFileAndDirectory.md) - Praktik terbaik untuk
  penamaan file dan direktori
- [Setup PHP](Bagian/-SetupPHP.md) - Panduan instalasi dan konfigurasi PHP
- [Setup XDebug PHP](Bagian/-SetupXDebugPHP.md) - Pengaturan XDebug untuk debugging PHP

Nanti akan ada bagian-bagian yang lain jika memang kontennya banyak.

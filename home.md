# Info Programming - Dokumentasi Teknologi Informasi

Selamat datang di Info Programming! Repositori ini berisi berbagai dokumentasi, cheat sheet, dan panduan tentang berbagai aspek teknologi informasi. Tujuan dari project ini adalah untuk menyediakan referensi yang berguna bagi mereka yang ingin mempelajari atau mengingat kembali berbagai konsep, perangkat lunak, dan teknik dalam dunia TI.

## Isi Dokumentasi

Project ini mencakup berbagai topik teknologi informasi, termasuk:

- Panduan penggunaan dan optimasi perangkat lunak
- Cheat sheet perintah dan konfigurasi
- Panduan konfigurasi dan praktik terbaik
- Tips dan trik untuk berbagai aplikasi dan sistem

## Bagian Dokumentasi

- **Bagian 1**: Berisi panduan-panduan dasar seperti upgrade LibreOffice, cheat sheet Ghostscript, panduan EditorConfig, dan penamaan file dan direktori.

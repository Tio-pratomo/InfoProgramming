# Dokumentasi Bagian 1 - Info Programming

Selamat datang di Bagian 1 dari dokumentasi Info Programming! Bagian ini berisi berbagai panduan dan dokumentasi penting terkait teknologi informasi.

## Daftar Isi

Berikut adalah daftar dokumentasi yang tersedia di Bagian 1:

- **Upgrade LibreOffice**: Panduan lengkap untuk melakukan upgrade versi LibreOffice
- **Ghostscript Cheat Sheet**: Kumpulan perintah-perintah penting untuk menggunakan Ghostscript
- **EditorConfig Guide**: Panduan konfigurasi EditorConfig untuk memastikan konsistensi format kode
- **Naming File and Directory**: Panduan praktik terbaik dalam memberi nama file dan direktori

## Tentang Dokumentasi Ini

Dokumentasi ini disusun untuk membantu pengguna dalam memahami dan menggunakan berbagai teknologi serta alat dalam dunia teknologi informasi. Setiap dokumentasi disusun secara praktis dengan contoh-contoh yang dapat langsung diterapkan dalam kehidupan nyata.

Silakan jelajahi dokumentasi yang tersedia untuk mendapatkan informasi yang Anda butuhkan.
